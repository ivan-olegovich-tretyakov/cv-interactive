# Ivan Tretyakov - Interactive CV Project

## Overview
This project aims to create a "Living CV" for Ivan Tretyakov, a senior RGM leader in the CPG/FMCG sector.
The CV will be an interactive HTML/Tailwind/JS page that allows recruiters to toggle between different professional personas:
1.  **The RGM Expert:** Focus on P&L, Price Pack Architecture, and Trade Investment.
2.  **The Digital Transformer:** Focus on AI-driven pricing, TPM systems, and automation.
3.  **The Strategic Leader:** Focus on 3-year plans, team leadership, and change management.

## Tech Stack
- HTML5
- Tailwind CSS (via CDN for portability)
- Vanilla JavaScript (for interactivity)
- Print-friendly CSS rules
